# Telegram Store Bot

بوت متجر تيليجرام كامل نسبيًا وجاهز للبداية على Railway مع:
- اشتراك إجباري بالقناة
- تحقق بشري بسؤال رياضي
- متجر منتجات
- مخزون أكواد
- شراء حسب الكمية
- رفع أكواد TXT من المدير
- محفظة ورصيد
- نظام دعوات ومكافأة بعد عدد محدد

## 1) التثبيت محليًا
```bash
npm install
npx prisma generate
npx prisma db push
npm start
```

## 2) متغيرات البيئة
انسخ `.env.example` إلى `.env` ثم عدل:
- `BOT_TOKEN`
- `DATABASE_URL`
- `CHANNEL_USERNAME` أو `CHANNEL_ID`
- `CHANNEL_URL`
- `ADMIN_IDS`

## 3) مهم جدًا
حتى يشتغل الاشتراك الإجباري:
- أضف البوت داخل القناة
- خله Admin
- إذا القناة عامة استخدم `CHANNEL_USERNAME=@channelname`
- وإذا خاصة تقدر تستخدم `CHANNEL_ID`

## 4) أوامر واستخدام
- `/start` دخول المستخدم
- `/admin` لوحة المدير

## 5) إضافة منتج
من لوحة المدير:
- `➕ إضافة منتج`
- الاسم
- السعر
- الوصف

## 6) رفع أكواد
- اضغط `📥 رفع أكواد`
- اختر المنتج
- أرسل ملف TXT
- كل سطر داخل الملف = كود واحد

مثال ملف:
```txt
CODE-111
CODE-222
CODE-333
```

## 7) الشراء
المستخدم:
- يدخل `🛍 المتجر`
- يختار المنتج
- يضغط شراء
- يرسل الكمية
- إذا رصيده كافي والمخزون موجود يستلم الأكواد مباشرة

## 8) Railway
بعد رفع المشروع:
```bash
npm install
npx prisma generate
npx prisma db push
npm start
```

في Railway:
- أنشئ Service للبوت
- أضف PostgreSQL
- انسخ `DATABASE_URL`
- أضف بقية المتغيرات
- Start Command:
```bash
npm start
```

## 9) ملاحظات
- الجلسات الحالية داخل الذاكرة، مناسبة كبداية.
- إذا تريد نسخة أقوى لاحقًا، حول الحالات إلى قاعدة البيانات.
- المشروع هذا جاهز كبداية قوية وقابل للتوسعة.
